LIRI Bot

Liri is an app run in node.js which searches through Bands-in-Town API, Spotify API, and OMDB API to retrieve information related to requests input by a user in terminal. It maintains a log of all commands and results in log.txt, utilises .gitignore and .env to store API keys, and can retrieve a line of information stored in random.txt to conduct a search through the APIs using the information stored there.

Technologies Used: JavaScript, Node.js, Axios NPM, DOTENV NPM, FS NPM, Inquirer NPM, Moment NPM, Request NPM, Bands In Town API, OMDB API, Spotify API
Core Concepts: Dynamic API Access, DOTENV
